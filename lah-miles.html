LAH 마일즈 시스템

프로젝트

LAH(소형무장헬기)에 육군 과학화 전투 훈련단에서 운용하는 교전 훈련 장비를 구매, 개조, 설치하여 KCTC 훈련장에서 운용이 가능하도록 LAH용 마일즈 장비를 개발하는 체계 개발 사업.

프로젝트명
LAH(Light Armed Helicopter) 마일즈 시스템 프로젝트
주사업자/발주처
한국항공우주산업 / 방위사업청
수행기간/사업규모
2017.03 ~ 2019.06 / 120억
역할
LAH(공격형 무장 헬기) 연동 시뮬레이터, S/W 개발 훈련통제SW 개발

프로젝트 개요

LAH(소형무장헬기)에 육군 과학화 전투 훈련단에서 운용하는 교전 훈련 장비를 구매, 개조, 설치하여 KCTC 훈련장에서 운용이 가능하도록 LAH용 마일즈 장비를 개발하는 체계 개발 사업.

담당역할

  • LAH(공격형 무장 헬기) 연동 시뮬레이터, S/W 개발 훈련통제SW 개발

시뮬레이터 S/W 개발

  • LAH와 마일즈 시스템 간 시험 데이터 송수신을 지원하는 로그 데이터 처리기(시뮬레이터 SW) 개발

훈련통제SW ↔ LAH 마일즈 연동

  • KCTC 훈련통제 S/W를 개조하여, LAH의 터렛화기 발사 정보 및 LAH 탑재 무기(공대지 유도탄, 무유도 로켓)의 구성 모의 기능 구현

국방 표준 산출물 작성

  • 방위사업청 지침에 따른 CDRL 산출물 작성
  • 작성 문서 범위: SRS, SDD, DBDD, IDD, STP, STD, STR 등

PDR / CDR / TRR 수행

  • 프로젝트 단계별 PDR(예비 설계 검토), CDR(상세 설계 검토), TRR(시험 준비 검토) 수행
  • 요구사항, 설계, 시험 준비 상태 점검

시험 및 품질 검증

  • 항공기 탑재 소프트웨어 신뢰성 표준 DO-178B에 해당하는 품질프로세스 수행 및 산출물 작성
  • 신뢰성 시험 수행
  • 정적 테스트: 코딩 규칙 검증, 실행시간 오류(CWE-660) 검출
  • 동적 테스트: Statement, Branch, MC/DC 기준 테스트

LAH 연동 테스트 수행

  • 소형무장헬기(LAH) ↔ 스마트다기능시현기(SMFD) ↔ 마일즈(MILES) ↔ 훈련통제 SW 간 터렛 직사화기 및 공대지 유도탄/무유도 로켓 모의 데이터 연동 검증
  • 초도 비행 시험 환경에서 연계 데이터 검증

DT/OT/초도비행 시험 수행

  • 소형무장헬기(LAH) 시제 1호기 초도비행 시 개발된 소형무장헬기(LAH) ↔ 스마트다기능시현기(SMFD) ↔ 마일즈(MILES) ↔ 훈련통제 SW 간 연동 요구 기능 성공적으로 검증 완료

관련 기술

언어 / 프레임워크

  • 언어: Java, SQL, JavaScript, JSP
  • F/W: Spring F/W, Netty F/W

플랫폼 / 미들웨어 / DB

  • O/S: Linux
  • WAS: JEUS, WebtoB
  • D/B: Oracle

도구

  • Tool: Eclipse, Miplatform, Sparrow, Toad for Oracle
