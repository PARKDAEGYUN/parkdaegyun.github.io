KCTC 체계 관련 이미지

프로젝트 개요

육군 과학화 전투훈련 중앙통제장비체계(KCTC)의 기존 대대급 규모 훈련장을 여단급 규모 훈련장으로 체계 개발하는 사업.

담당역할

  • 통신 관리 서버 개발, 모의 기능 구현, 시험평가 수행, 시스템 운영

통신 관리 서버 개발

  • Netty Framework를 활용하여 훈련장 중계기 서버↔훈련통제S/W 서버 간 데이터 송·수신 처리를 담당하는 통신 관리 서버 S/W 개발

모의 기능 구현

  • 지뢰지대, 곡사화기 모의 데이터 생성 기능 구현
  • 모의 데이터 훈련장 전파 및 피해결과 수신 기능 개발

국방 표준 산출물 작성

  • 방위사업청 지침에 따른 CDRL 산출물 작성
  • 작성 문서 범위: SRS, SDD, DBDD, IDD, STP, STD, STR 등

신뢰성 시험 진행

  • 정적 테스트(코딩 규칙 검증)
  • 동적 테스트(Statement, Branch 기반) 테스트 케이스 설계 및 수행

체계 시험평가 수행

  • KCTC 체계 개발시험평가(DT) 및 운용시험평가(OT) 수행

시스템 운영

  • WAS(JEUS/WebtoB) 운영 관리로 안정적 서비스 운영 및 유지보수 지원

관련 기술

언어 / 프레임워크

  • 언어: Java, SQL, JavaScript
  • F/W: Spring F/W, Netty F/W

플랫폼 / 미들웨어 / DB

  • O/S: Linux
  • WAS: JEUS, WebtoB
  • D/B: Oracle

도구

  • Tool: Eclipse, Miplatform, Toad for Oracle, Sparrow
