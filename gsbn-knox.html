삼성 KNOX 소프트웨어

라이선스 발급

시스템 구축

삼성전자 GSBN 기반 웹 무역 포털 내 KNOX 소프트웨어 라이선스의 B2B 구매·정산 체계를 구축하고, S-ERP API 연계를 통해 주문·발급·빌링 처리를 수행하는 시스템 구축.

삼성 KNOX 소프트웨어 라이선스 발급, 빌링 시스템 운영

프로젝트 개요

삼성전자 GSBN(Global Samsung Business Network) 기반 웹 무역 포털 내 삼성 KNOX 소프트웨어 라이선스의 B2B 구매, 정산 체계를 구축하는 프로젝트로, S-ERP의 API 연계를 통해 기업 대상 라이선스 주문, 발급, 빌링 처리 시스템 구축 프로젝트.

담당역할

  • 삼성 KNOX 소프트웨어 라이선스 발급, 빌링 시스템 운영

삼성 KNOX 라이선스 발급 시스템 개발·운영

  • 삼성 갤럭시 스마트폰 보안 솔루션 KNOX 라이선스 발급, 빌링 처리 시스템 운영
  • GSBN 기반 삼성전자 웹 무역 포털 내 KNOX 구매 시스템 구축 및 S-ERP 연계
  • 등록된 거래선 대상 라이선스 주문, 발급 및 월간/연간 과금 처리 B2B 시스템 개발/운영

판매량 집계 자동화

  • 삼성 KNOX 소프트웨어 판매 데이터를 인터페이스로 일별 자동 수신·집계하는 배치/화면 개발
  • 집계 데이터를 활용한 통계 보고 및 분석 체계 지원

SQL 성능 개선

  • 비효율 쿼리 실행계획 분석·튜닝으로 데이터 처리 성능 개선
  • 화면↔WAS↔DB 전 구간 트랜잭션 흐름 추적 및 병목/지연 원인 분석
  • 응답 지연 SQL 식별 및 실행계획 분석·튜닝 수행
