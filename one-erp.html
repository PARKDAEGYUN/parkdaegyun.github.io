Project Case Study

한화에어로스페이스 차세대 One-ERP 경영지원(BSP:Business Support Program) 모듈

통합 ERP 중심의 차세대 시스템 구축에서 BSP 모듈 PL로 참여하여 범위·일정·자원·리스크를 관리하고, 표준문서/법무/개선업무 모듈의 분석·설계·개발과 DB 설계·성능 최적화·마이그레이션·통합 테스트·안정화까지 주도.

프로젝트명
한화에어로스페이스 차세대 One-ERP 프로젝트
주사업자/발주처
한화시스템 / 한화에어로스페이스
수행기간/사업규모
2023.11 ~ 2025.04 / 726억
역할
경영지원(BSP) 모듈 개발, 리딩, DB 설계 및 성능 최적화

프로젝트 개요

한화디펜스·한화방산이 한화에어로스페이스로 통합법인 출범에 맞추어 추진한 차세대 시스템 구축 사업으로, 통합 ERP 시스템을 중심으로 MES, PLM, SRM, SCM, BSP, QIS, AM-AP, EHS 등 기존 Non-ERP·레거시 시스템을 표준화, 통합, 재구축한 프로젝트.

담당역할

  • 경영지원(BSP: Business Support Program) 모듈 개발, 리딩, DB 설계 및 성능 최적화

BSP 모듈 개발 리딩

  • 프로젝트 범위, 일정, 자원 계획 수립 및 WBS 작성
  • 6인 개발팀 리딩, 기능점검 코드 리뷰 및 품질 관리

BSP(경영지원) 모듈 개발

  • 표준문서관리, 법무관리, 개선업무관리, 배차관리, 사회공헌, 공문서관리, ITSM, 보안관리, 특허관리 등 전사 경영지원 시스템의 주요 기능 개발
  • 그룹웨어와 REST API 연동하여 전자결재 송수신 기능 개발
  • EAI 통해 ERP와 연동하여 개선 시상금, 특허 출원금 등 급여 반영기능 개발

산출물 작성 및 관리

  • 화면 설계서, 프로그램 설계서, 인터페이스 설계서, 배치 프로그램 설계서, 단위/통합테스트 결과서, 사용자매뉴얼, 운영자매뉴얼 등 산출물 작성 및 최신화 관리

데이터 마이그레이션

  • 기존 3사(에어로, 디펜스, 방산)의 레거시 시스템 DB에 존재하는 각 코드/데이터를 통합·표준화 처리 후, 신규 개발된 시스템으로 통합 이관 처리

SQL 성능 개선

  • 비효율이 높은 쿼리 실행계획 분석·튜닝하여 데이터 처리 성능 개선
  • 화면↔WAS↔DB 전 구간 트랜잭션 흐름 추적 및 병목 지점/지연 원인 분석
  • 응답 속도가 느린 비효율적 SQL 식별 및 실행 계획 분석·튜닝 수행

테스트 / 표준화 / 안정화

  • 신뢰성/단위/통합 테스트 수행을 통해 품질 및 표준화 기반 확보
  • 정적테스트 SPARROW 툴 활용하여 코딩 규칙 표준화 처리
  • 통합 테스트 시 현업 사용자 대상 테스트 수행, 오류/개선사항 반영
  • 각 산출물 및 결과물 납품 후 안정화 단계 모니터링 수행, 오픈 초기 안정성 확보
  • 시스템 오픈 이후 ITSM 통해 접수된 오류 수정, 개선사항, 데이터 수정 요청 반영
  • 오픈 이후 안정화 기간 긴급배포 등 신속 대응 체계 마련, 시스템 이중화로 안정적 서비스 제공
  • 프로그램 형상 변경 시 설계 산출물 최신화

성과 & 결과

  • 프로젝트 일정 내 성공적으로 시스템 오픈
  • 한화에어로스페이스 고객 만족도 조사 결과 90% 이상 만족으로 응답
  • 오픈/안정화 단계 이슈 대응 및 운영 안정성 확보(ITSM·긴급배포·이중화)

관련 기술

언어 / 프레임워크

  • 언어: Java, SQL, JavaScript
  • F/W: Spring F/W
  • Tool: Spring Boot, Nexacro

플랫폼 / 미들웨어 / DB

  • O/S: Linux
  • WAS: JEUS, WebtoB
  • D/B: Tibero / Oracle

도구

  • DBeaver, Jenkins, OZ Report, Easy PDF, Git, Sourcetree, ChakraMax, Hiware
